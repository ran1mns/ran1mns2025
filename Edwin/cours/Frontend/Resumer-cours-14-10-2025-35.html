Titre du Cours : HTML Basique

Introduction

Le HTML (HyperText Markup Language) est le langage de base pour créer des pages web. Il permet de structurer le contenu d'une page en utilisant des balises. Comprendre le HTML est essentiel pour toute personne souhaitant se lancer dans le développement web.

Concepts Clés

Structure de Base

Toute page HTML commence par une balise <html> qui englobe tout le document. Ensuite, nous avons la balise <head> pour les métadonnées et la balise <body> pour le contenu visible.


<!DOCTYPE html>
<html>
<head>
  <title>Titre de la page</title>
</head>
<body>
  <h1>Bienvenue sur ma page web</h1>
  <p>Ceci est un paragraphe.</p>
</body>
</html>

Balises de Titre

Les balises de titre, de <h1> à <h6>, sont utilisées pour structurer le contenu. <h1> est le titre principal, tandis que <h6> est le moins important.


<h1>Titre principal</h1>
<h2>Sous-titre</h2>
<h3>Sous-sous-titre</h3>

Paragraphes et Texte

Les paragraphes sont définis par la balise <p>. Pour le texte en ligne, on utilise <span>.


<p>Ceci est un paragraphe.</p>
<p>Voici un autre paragraphe avec du texte <span>en ligne</span>.</p>

Listes

Les listes peuvent être ordonnées (<ol>) ou non ordonnées (<ul>). Chaque élément de la liste est défini par <li>.


<ul>
  <li>Élément de liste non ordonnée</li>
  <li>Un autre élément</li>
</ul>

<ol>
  <li>Premier élément de liste ordonnée</li>
  <li>Deuxième élément</li>
</ol>

Lihens et Images

Les liens sont créés avec la balise <a> et les images avec <img>.


<a href="https://www.example.com">Cliquez ici</a>
<img src="image.jpg" alt="Description de l'image">

Bonnes Pratiques

Utilisez toujours des attributs alt pour les images afin d'améliorer l'accessibilité.

Exercices Pratiques

  1. Créez une page HTML avec un titre principal, deux paragraphes, une liste non ordonnée de trois éléments, et un lien vers votre site préféré.
Corrigé 

<!DOCTYPE html>
<html>
<head>
  <title>Mon Premier Exercice</title>
</head>
<body>
  <h1>Bienvenue sur ma page</h1>
  <p>Ceci est le premier paragraphe.</p>
  <p>Ceci est le deuxième paragraphe.</p>
  <ul>
    <li>Élément 1</li>
    <li>Élément 2</li>
    <li>Élément 3</li>
  </ul>
  <a href="https://www.example.com">Mon site préféré</a>
</body>
</html>
