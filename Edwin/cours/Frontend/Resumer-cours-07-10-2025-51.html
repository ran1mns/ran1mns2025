Titre du Cours : Les Fonctions en JavaScript

Introduction

Les fonctions sont des blocs de code réutilisables qui effectuent une tâche spécifique. En JavaScript, elles sont essentielles pour organiser et structurer le code. Comprendre et maîtriser les fonctions vous permettra de créer des programmes plus efficaces et maintenables.

Concepts Clés

Définition et Syntaxe de Base

Une fonction en JavaScript est définie à l'aide du mot-clé function suivi du nom de la fonction, de parenthèses et d'un bloc de code entre accolades. Voici un exemple simple :


function direBonjour() {
  console.log("Bonjour!");
}

Appel de Fonction

Pour exécuter une fonction, vous devez l'appeler en utilisant son nom suivi de parenthèses. Par exemple :


direBonjour(); // Affiche "Bonjour!" dans la console

Paramètres et Arguments

Les fonctions peuvent accepter des paramètres, qui sont des valeurs d'entrée. Lorsque vous appelez une fonction, vous passez des arguments correspondant à ces paramètres. Voici un exemple :


function saluer(nom) {
  console.log("Bonjour, " + nom + "!");
}

saluer("Alice"); // Affiche "Bonjour, Alice!" dans la console

Valeur de Retour

Les fonctions peuvent retourner une valeur en utilisant le mot-clé return. Cette valeur peut ensuite être utilisée ou stockée. Par exemple :


function additionner(a, b) {
  return a + b;
}

let resultat = additionner(3, 4); // resultat vaut 7
console.log(resultat); // Affiche 7 dans la console

Fonctions Fléchées

Introduites avec ES6, les fonctions fléchées offrent une syntaxe plus concise. Voici comment définir une fonction fléchée :


const direBonjour = () => {
  console.log("Bonjour!");
};

direBonjour(); // Affiche "Bonjour!" dans la console

Portée des Variables

Les variables définies à l'intérieur d'une fonction sont locales à cette fonction et ne sont pas accessibles en dehors. Voici un exemple :


function exemplePortee() {
  let message = "Bonjour!";
  console.log(message); // Affiche "Bonjour!"
}

exemplePortee();
console.log(message); // Erreur : message n'est pas défini

Astuce

Utilisez des noms de fonctions descriptifs pour rendre votre code plus lisible et maintenable.

Exercices Pratiques

  1. Créez une fonction qui prend deux nombres en paramètres et retourne leur produit.
  2. Créez une fonction fléchée qui prend un nom en paramètre et affiche un message de bienvenue personnalisé.
Corrigé 

// Exercice 1
function multiplier(a, b) {
  return a * b;
}

let resultat = multiplier(5, 6); // resultat vaut 30
console.log(resultat); // Affiche 30 dans la console

// Exercice 2
const saluer = (nom) => {
  console.log("Bienvenue, " + nom + "!");
};

saluer("Bob"); // Affiche "Bienvenue, Bob!" dans la console
