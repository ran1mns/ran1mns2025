Titre du Cours : HTML Basique

Introduction

Le HTML, ou HyperText Markup Language, est le langage de base utilisé pour créer des pages web. Il permet de structurer le contenu d'une page en utilisant des balises spécifiques. Apprendre le HTML est la première étape pour devenir un développeur web.

Concepts Clés

Structure de Base d'un Document HTML

Tout document HTML commence par une déclaration de type de document (doctype) suivie des balises <html>, <head> et <body>. Voici un exemple de structure de base :


<!DOCTYPE html>
<html>
  <head>
    <title>Titre de la page</title>
  </head>
  <body>
    <h1>Bienvenue sur ma page web</h1>
    <p>Ceci est un paragraphe.</p>
  </body>
</html>

Balises de Titre

Les balises de titre, de <h1> à <h6>, sont utilisées pour définir les titres et sous-titres dans un document HTML. <h1> est le plus important et <h6> le moins important.


<h1>Titre principal</h1>
<h2>Sous-titre</h2>
<h3>Sous-sous-titre</h3>

Balises de Paragraphe

La balise <p> est utilisée pour définir un paragraphe de texte. Chaque paragraphe est automatiquement séparé par un espace.


<p>Ceci est un paragraphe.</p>
<p>Ceci est un autre paragraphe.</p>

Balises de Liste

Il existe deux types de listes en HTML : les listes ordonnées (<ol>) et les listes non ordonnées (<ul>). Chaque élément de la liste est défini par la balise <li>.


<!-- Liste non ordonnée -->
<ul>
  <li>Élément 1</li>
  <li>Élément 2</li>
</ul>

<!-- Liste ordonnée -->
<ol>
  <li>Élément 1</li>
  <li>Élément 2</li>
</ol>

Balises de Lien

La balise <a> est utilisée pour créer des liens hypertextes. L'attribut href spécifie l'URL de la page vers laquelle le lien pointe.


<a href="https://www.example.com">Visitez Example.com</a>

Balises d'Image

La balise <img> est utilisée pour insérer une image dans une page web. Les attributs src et alt sont obligatoires.


<img src="chemin/vers/image.jpg" alt="Description de l'image">

Astuce

Utilisez toujours l'attribut alt pour décrire l'image. Cela améliore l'accessibilité de votre site pour les personnes utilisant des lecteurs d'écran.
