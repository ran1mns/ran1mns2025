Introduction au HTML

HTML (HyperText Markup Language) est le langage standard pour créer et structurer le contenu des pages web. Il fonctionne à l'aide d'un système de balises qui indiquent au navigateur comment afficher les différents éléments comme les titres, les paragraphes, les images, etc.

1. Structure d'un Document HTML

Chaque page HTML suit une structure de base :


<!DOCTYPE html>
<html lang="fr">
  <head>
    <!-- Les méta-informations vont ici -->
    <meta charset="UTF-8">
    <title>Titre de la page</title>
  </head>
  <body>
    <!-- Le contenu visible de la page va ici -->
  </body>
</html>
  • <!DOCTYPE html> : Définit le type de document.
  • <html> : L'élément racine de la page.
  • <head> : Contient les méta-informations (titre, encodage, liens CSS...).
  • <body> : Contient le contenu visible de la page.

2. Balises de Contenu Courantes

Titres et Paragraphes

<h1> à <h6> pour les titres (du plus au moins important) et <p> pour les paragraphes.

Listes

<ul> pour les listes à puces (non ordonnées) et <ol> pour les listes numérotées (ordonnées). Chaque élément de liste est défini par <li>.

Liens (Hyperliens)

La balise <a> crée un lien. L'attribut href contient l'URL de destination. On ajoute souvent target="_blank" pour ouvrir le lien dans un nouvel onglet.


<a href="https://www.wikipedia.org" target="_blank">Visiter Wikipédia</a>

Images

La balise <img> insère une image. L'attribut src contient le chemin vers l'image et l'attribut alt fournit un texte alternatif essentiel pour l'accessibilité.


<img src="images/logo.png" alt="Logo de mon site" width="150">

3. HTML Sémantique (HTML5)

HTML5 a introduit des balises sémantiques qui décrivent mieux la structure de la page. Leur utilisation est une bonne pratique pour le référencement (SEO) et l'accessibilité.

  • <header> : Pour l'en-tête de la page (logo, titre principal, navigation).
  • <nav> : Pour le menu de navigation principal.
  • <main> : Pour le contenu principal et unique de la page.
  • <section> : Pour regrouper un contenu thématique.
  • <article> : Pour un contenu autonome (un article de blog, un produit).
  • <footer> : Pour le pied de page (informations de contact, liens annexes).

Exercice Pratique

Créez une page HTML simple sur votre animal préféré en utilisant les balises sémantiques. La page doit contenir :

  1. Un header avec un titre <h1>.
  2. Un main contenant :
    • Une section avec un titre <h2> et une image de l'animal.
    • Une autre section avec un titre <h2> et une liste <ul> de ses caractéristiques.
  3. Un footer avec un paragraphe <p> contenant un lien vers une page d'information (ex: Wikipédia).
Voir un exemple de structure 

<!DOCTYPE html>
<html lang="fr">
  <head>
    <title>Le Panda Roux</title>
  </head>
  <body>
    <header>
      <h1>Le fascinant Panda Roux</h1>
    </header>
    <main>
      <section>
        <h2>Qui est-il ?</h2>
        <img src="panda.jpg" alt="Un panda roux dans un arbre">
      </section>
      <section>
        <h2>Caractéristiques</h2>
        <ul>
          <li>Pelage roux et doux</li>
          <li>Longue queue touffue</li>
          <li>Se nourrit principalement de bambou</li>
        </ul>
      </section>
    </main>
    <footer>
      <p>Pour en savoir plus, visitez la page <a href="..." target="_blank">Wikipédia</a>.</p>
    </footer>
  </body>
</html>
