Attribut ALT pour les images

L'attribut alt est utilisé pour remplacer l'image par un texte :

  • En cas d'erreur de chargement de l'image
  • Pour l'accessibilité, uniquement si l'information n'est pas présente ailleurs
Remarque : alt est obligatoire, même vide.

CSS pour le design

Avant de positionner les éléments, commencez par appliquer des couleurs :

  • Le positionnement peut se faire via float (obsolète mais encore utile à connaître)
  • Flexbox et Grid sont les méthodes modernes et recommandées

Début du CSS

Différentes façons d'ajouter du CSS :

  • Inline : style="color: brown;" (doit être répété sur chaque élément)
  • Dans le <head> avec une balise <style> ... </style> :
    • <style>
h1 { color: red; }
</style>
  • Fichier externe style.css : recommandé pour centraliser les styles :
    • <link rel="stylesheet" href="style.css">

Les styles peuvent être appliqués selon la hiérarchie DOM et la spécificité des sélecteurs :

  • Plus le sélecteur est précis, plus il a de poids
  • Exemple : .class balise > balise applique le style uniquement aux enfants de .class

Classes et IDs

  • Les id="" sont uniques (comme une plaque d'immatriculation)
  • En CSS, on utilise #id
  • Utile pour JavaScript, peu recommandé pour CSS moderne (souvent considéré comme un "refuge")

Balises sémantiques

Utilisation des balises pour structurer correctement la page :

  • <header>, <nav>, <main>, <footer>
  • <section>, <article>, <aside>
