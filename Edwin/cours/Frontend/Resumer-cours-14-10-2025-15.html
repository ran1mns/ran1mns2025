Titre du Cours : CSS Flexbox

Introduction

CSS Flexbox est un modèle de mise en page qui permet de créer des dispositions complexes de manière simple et efficace. Il est particulièrement utile pour créer des interfaces utilisateur réactives et adaptables à différentes tailles d'écran. Flexbox permet de contrôler la disposition des éléments enfants d'un conteneur flexible, facilitant ainsi l'alignement et la distribution de l'espace.

Concepts Clés

Le Conteneur Flexible

Pour utiliser Flexbox, vous devez d'abord définir un conteneur flexible en utilisant la propriété display: flex;. Ce conteneur devient un "flex container" et tous ses enfants directs deviennent des "flex items".


      <div class="flex-container">
        <div class="flex-item">Item 1</div>
        <div class="flex-item">Item 2</div>
        <div class="flex-item">Item 3</div>
      </div>
    

      .flex-container {
        display: flex;
      }

Direction des Flex Items

La propriété flex-direction permet de définir la direction principale des éléments flexibles. Les valeurs possibles sont row (par défaut), row-reverse, column et column-reverse.


      .flex-container {
        display: flex;
        flex-direction: column; /* Dispose les éléments en colonne */
      }

Alignement des Flex Items

La propriété justify-content permet d'aligner les éléments flexibles le long de l'axe principal. Les valeurs possibles incluent flex-start, flex-end, center, space-between, space-around et space-evenly.


      .flex-container {
        display: flex;
        justify-content: space-between; /* Espacement égal entre les éléments */
      }

La propriété align-items permet d'aligner les éléments flexibles le long de l'axe transversal. Les valeurs possibles incluent flex-start, flex-end, center, baseline et stretch.


      .flex-container {
        display: flex;
        align-items: center; /* Centre les éléments verticalement */
      }

Ordre des Flex Items

La propriété order permet de changer l'ordre des éléments flexibles. Par défaut, tous les éléments ont un ordre de 0. Les éléments avec un ordre plus élevé apparaîtront après ceux avec un ordre plus bas.


      .flex-item:nth-child(2) {
        order: 1; /* Déplace le deuxième élément à la fin */
      }

Flexibilité des Flex Items

Les propriétés flex-grow, flex-shrink et flex-basis permettent de contrôler la flexibilité des éléments flexibles.

  • flex-grow : Définit la capacité d'un élément à croître par rapport aux autres éléments.
  • flex-shrink : Définit la capacité d'un élément à rétrécir par rapport aux autres éléments.
  • flex-basis : Définit la taille initiale d'un élément avant que l'espace restant ne soit distribué.

      .flex-item {
        flex-grow: 1; /* Permet aux éléments de croître de manière égale */
        flex-shrink: 1; /* Permet aux éléments de rétrécir de manière égale */
        flex-basis: 100px; /* Taille initiale de 100px */
      }

Bonnes Pratiques

Utilisez Flexbox pour créer des dispositions simples et réactives. Évitez d'utiliser Flexbox pour des dispositions complexes ou imbriquées, car cela peut rendre le code difficile à maintenir.

Erreurs Fréquentes

Une erreur courante est d'oublier de définir un conteneur flexible avec display: flex;. Sans cette propriété, les éléments enfants ne seront pas traités comme des éléments flexibles.

Exercices Pratiques

  1. Créez un conteneur flexible avec trois éléments enfants. Alignez les éléments horizontalement et centrez-les verticalement.
