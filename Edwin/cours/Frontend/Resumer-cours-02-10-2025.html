Titre du Cours : Cours CSS – 02 Octobre 2025

Fonts et Comic

  • font-family : permet de définir la police d’écriture
  • Ne jamais utiliser "Comic" : utilisée à outrance, usage péjoratif

Propriété !important

Exemple :

color: red !important;

Cette propriété écrase tous les autres styles et est une rustine efficace mais non durable. Elle indique généralement une difficulté à trouver le sélecteur correct.

Maquette et structure HTML

  • Commencer par une maquette simple (pas de responsive pour le moment)
  • Dégager les balises structurantes ou landmarks / sémantiques : header, main, footer, nav, section, aside
  • Définir les listes de navigation : ul et li dans le nav
  • Créer un dossier de base, un sous-dossier css avec style.css, et l’index.html obligatoire
  • Donner temporairement une couleur de fond aux balises pour visualiser les espaces

CSS et sélection des éléments

Pour cibler précisément les éléments :

header nav > ul { ... }

Le symbole > indique que le ul doit être un enfant direct de nav.
