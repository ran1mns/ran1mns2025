Titre du Cours : HTML Basique

Introduction

L'HTML (HyperText Markup Language) est le langage de base pour créer des pages web. Il permet de structurer le contenu d'une page en utilisant des balises. Apprendre l'HTML est essentiel pour toute personne souhaitant créer des sites web, car il est la fondation sur laquelle reposent le CSS et le JavaScript.

Concepts Clés

Les Balises HTML

Les balises HTML sont des éléments qui permettent de structurer le contenu d'une page web. Chaque balise a une fonction spécifique. Par exemple, la balise <h1> est utilisée pour les titres principaux, tandis que la balise <p> est utilisée pour les paragraphes.


<!DOCTYPE html>
<html>
<head>
  <title>Ma Première Page</title>
</head>
<body>
  <h1>Bienvenue sur ma page</h1>
  <p>Ceci est un paragraphe.</p>
</body>
</html>

Les Attributs HTML

Les attributs fournissent des informations supplémentaires sur les éléments HTML. Par exemple, l'attribut href dans une balise <a> spécifie l'URL de la page à laquelle le lien pointe.


<a href="https://www.example.com">Visitez Example</a>

Les Éléments de Liste

Les listes permettent de regrouper des éléments similaires. Il existe deux types principaux de listes : les listes ordonnées (<ol>) et les listes non ordonnées (<ul>).


<ul>
  <li>Élément 1</li>
  <li>Élément 2</li>
</ul>

<ol>
  <li>Premier élément</li>
  <li>Deuxième élément</li>
</ol>

Les Images

Pour ajouter des images à votre page web, utilisez la balise <img>. L'attribut src spécifie le chemin de l'image, et l'attribut alt fournit un texte alternatif pour les navigateurs qui ne peuvent pas afficher l'image.


<img src="chemin/vers/image.jpg" alt="Description de l'image">

Les Formulaires

Les formulaires permettent aux utilisateurs d'entrer des données. La balise <form> est utilisée pour créer un formulaire, et divers éléments comme <input>, <textarea>, et <button> sont utilisés pour capturer les données.


<form action="/submit" method="post">
  <label for="name">Nom :</label>
  <input type="text" id="name" name="name">
  <button type="submit">Envoyer</button>
</form>

Exercices Pratiques

  1. Créez une page HTML avec un titre principal, un paragraphe, une liste non ordonnée de trois éléments, et une image avec un texte alternatif.
  2. Ajoutez un formulaire à votre page avec un champ de texte pour le nom, un champ de texte pour l'email, et un bouton d'envoi.
Corrigé 

<!DOCTYPE html>
<html>
<head>
  <title>Exercice Pratique</title>

    <li>Élément 2</li>

  </ul>

    <label for="email">Email :</label>
    <input type="email" id="email" name="email">
    <button type="submit">Envoyer</button>
  </form>
</body>
</html>
