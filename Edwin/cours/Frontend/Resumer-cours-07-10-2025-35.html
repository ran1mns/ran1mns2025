Titre du Cours : HTML Basique

Introduction

L'HTML (HyperText Markup Language) est le langage de base pour créer des pages web. Il permet de structurer le contenu d'une page en utilisant des balises. Apprendre l'HTML est essentiel pour tout développeur web, car c'est la fondation sur laquelle reposent les autres technologies comme le CSS et JavaScript.

Concepts Clés

Structure de Base d'un Document HTML

Tout document HTML commence par une déclaration de type de document (doctype) suivie de la balise <html>. À l'intérieur de cette balise, nous avons deux sections principales : <head> et <body>.


<!DOCTYPE html>
<html>
  <head>
    <title>Mon Premier Site Web</title>
  </head>
  <body>
    <h1>Bienvenue sur mon site</h1>
    <p>Ceci est un paragraphe.</p>
  </body>
</html>

Balises de Titre

Les balises de titre (<h1> à <h6>) sont utilisées pour définir les titres et sous-titres. <h1> est le plus important et <h6> le moins important.


<h1>Titre Principal</h1>
<h2>Sous-titre</h2>
<h3>Sous-sous-titre</h3>

Paragraphes et Texte

La balise <p> est utilisée pour définir un paragraphe de texte.


<p>Ceci est un paragraphe de texte.</p>

Listes

Il existe deux types de listes en HTML : les listes ordonnées (<ol>) et les listes non ordonnées (<ul>). Chaque élément de la liste est défini par la balise <li>.


<!-- Liste ordonnée -->
<ol>
  <li>Élément 1</li>
  <li>Élément 2</li>
</ol>

<!-- Liste non ordonnée -->
<ul>
  <li>Élément 1</li>
  <li>Élément 2</li>
</ul>

Liens Hypertexte

La balise <a> est utilisée pour créer des liens hypertexte. L'attribut href spécifie l'URL de la page de destination.


<a href="https://www.example.com">Visitez Example.com</a>

Images

La balise <img> est utilisée pour insérer des images. L'attribut src spécifie le chemin de l'image, et l'attribut alt fournit une description textuelle de l'image.


<img src="chemin/vers/image.jpg" alt="Description de l'image">

Bonnes Pratiques

  • Utilisez toujours des balises sémantiques pour améliorer l'accessibilité et le référencement.
  • Fermez toujours vos balises pour éviter les erreurs de rendu.
  • Utilisez des commentaires pour rendre votre code plus lisible.

Erreurs Fréquentes

  • Oublier de fermer une balise.
  • Utiliser des balises obsolètes comme <font>.
  • Ne pas fournir d'attribut alt pour les images, ce qui nuit à l'accessibilité.

Exercices Pratiques
