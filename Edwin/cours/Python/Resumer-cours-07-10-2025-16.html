Titre du Cours : Introduction à Python

Introduction

Python est un langage de programmation simple et puissant, créé en 1991 par Guido van Rossum. Il permet de créer des scripts rapidement sans nécessiter une infrastructure lourde. Python est particulièrement apprécié pour sa syntaxe claire et sa facilité d'apprentissage. Il est souvent utilisé pour des tâches variées, allant de l'analyse de données à l'automatisation de tâches en passant par le développement web.

Concepts Clés

Interaction avec Python

Vous pouvez interagir avec Python de deux manières principales : via un script ou directement dans le terminal. Pour écrire du code Python dans le terminal, il suffit de taper python suivi de votre code.


python
print("Hello, World!")

Manipulation des Données Chiffrées

Python permet de manipuler des données chiffrées facilement. Par exemple, pour calculer une puissance, vous utilisez l'opérateur **.


2 ** 3  # Cela signifie 2 puissance 3, soit 8

Python respecte les règles classiques des mathématiques, y compris l'utilisation des parenthèses pour définir l'ordre des opérations.


(2 + 3) * 4  # Cela signifie (2 + 3) multiplié par 4, soit 20

Concaténation de Chaînes de Caractères

En Python, vous pouvez ajouter des chaînes de caractères pour les concaténer.


"Hello, " + "World!"  # Cela donne "Hello, World!"

Vous pouvez utiliser des guillemets simples ou doubles pour définir une chaîne de caractères. Par convention, on utilise les guillemets doubles.


"j'ai faim"  # Utilisation de guillemets doubles
'j\'ai faim'  # Utilisation de guillemets simples avec échappement

Booléens

Python utilise les booléens True et False (attention à la majuscule).


is_hungry = True
is_full = False

Constantes

Python ne gère pas les constantes de manière stricte. Par convention, on utilise des majuscules pour les nommer, mais elles restent modifiables.


PI = 3.14
PI = 3.14159  # La constante peut être modifiée

None et Chaîne Vide

Il est important de distinguer entre None (rien) et une chaîne vide "".


empty_string = ""
no_value = None

Conversion de Types

Python ne peut pas traiter plusieurs types de données en même temps. Vous devez forcer la conversion des données en un type commun.


age = 25
age_str = str(age)  # Conversion en chaîne de caractères

Les f-strings permettent de formatter des chaînes de caractères en incluant des variables.


name = "Alice"
age = 30
message = f"{name} a {age} ans."

Conventions de Nommage

Il existe plusieurs conventions de nommage en Python :

  • Snake Case : rounded_average
  • Camel Case : roundedAverage
  • Pascal Case : RoundedAverage (usage spécifique en Python)

Nommage des Variables

Soyez attentif au nommage des variables. Évitez de commencer par un chiffre ou une majuscule, et n'utilisez pas de caractères spéciaux.


valid_variable = 10
invalid-variable = 20  # Invalide à cause du tiret

Utiliser is, has, ou can dans le nom d'une variable peut indiquer une condition.


is_valid = True
has_permission = False

Listes (ou Tableaux)

Une liste est un élément ordonné. Vous pouvez manipuler les éléments de manière groupée ou individuellement.


fruits = ["apple", "banana", "cherry"]
print(fruits[0])  # Accès au premier élément
