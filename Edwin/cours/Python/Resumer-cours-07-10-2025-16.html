Titre du Cours : Introduction à Python

Introduction

Python est un langage de programmation simple et puissant, créé en 1991 par Guido van Rossum. Il permet de créer des scripts rapidement sans nécessiter une infrastructure lourde. Python est particulièrement apprécié pour sa syntaxe claire et sa facilité d'apprentissage. Il est souvent utilisé pour des tâches variées, allant de l'analyse de données à l'automatisation de tâches en passant par le développement web.

Concepts Clés

Interaction avec Python

Vous pouvez interagir avec Python de deux manières principales : via un script (un fichier avec l'extension .py) ou directement dans le terminal (l'interpréteur interactif).


# Lancer l'interpréteur interactif
python

# Écrire du code directement
>>> print("Hello, World!")
Hello, World!
>>> exit()

Types de Données

Python supporte plusieurs types de données de base :

  • Nombres : int (entiers), float (nombres à virgule flottante).
  • Texte : str (chaînes de caractères).
  • Booléens : bool (True ou False).
  • Collections : list (listes), tuple, dict (dictionnaires).

# Exemples de types de données
age = 25               # int
pi = 3.14              # float
nom = "Edwin"          # str
est_majeur = True      # bool
fruits = ["pomme", "banane"] # list

Opérateurs

Python utilise des opérateurs pour effectuer des opérations sur les variables et les valeurs.

  • Arithmétiques : +, -, *, /, % (modulo), ** (puissance).
  • De comparaison : ==, !=, >, <, >=, <=.
  • Logiques : and, or, not.

Structures de Contrôle : Conditions

Les conditions permettent d'exécuter du code uniquement si une certaine condition est remplie. On utilise if, elif (sinon si), et else (sinon).


age = 18
if age >= 18:
    print("Vous êtes majeur.")
elif age < 0:
    print("Âge invalide.")
else:
    print("Vous êtes mineur.")

Structures de Contrôle : Boucles

Les boucles permettent de répéter un bloc de code plusieurs fois.

Boucle for

La boucle for est utilisée pour itérer sur une séquence (comme une liste).


fruits = ["pomme", "banane", "cerise"]
for fruit in fruits:
    print(fruit)

Boucle while

La boucle while s'exécute tant qu'une condition est vraie.


compteur = 0
while compteur < 5:
    print(compteur)
    compteur += 1  # Équivalent à compteur = compteur + 1

Fonctions

Une fonction est un bloc de code réutilisable qui effectue une action spécifique. On la définit avec le mot-clé def.


def saluer(nom):
    return f"Bonjour, {nom} !"

message = saluer("Edwin")
print(message)  # Affiche "Bonjour, Edwin !"
