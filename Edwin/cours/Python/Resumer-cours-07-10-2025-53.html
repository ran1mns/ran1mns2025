Nous aborderons des concepts clés comme les boucles, les conditions, et les structures de données.

Concepts Clés

Qu'est-ce qu'un Algorithme ?

Un algorithme est une suite d'instructions finies et non ambiguës permettant de résoudre un problème. Chaque problème peut être résolu de multiples façons, et l'objectif est de trouver une solution efficace et claire.

Représentation d'un Algorithme

On peut représenter un algorithme de deux manières principales :

  • Le pseudo-code : Un langage de description d'algorithmes qui ressemble à un langage de programmation mais qui est destiné à être lu par des humains.
  • Le logigramme (ou organigramme) : Une représentation graphique de l'algorithme avec des formes standardisées pour chaque type d'opération (début/fin, instruction, condition, entrée/sortie).

Exemple : "Devine le nombre"

Voici un exemple d'algorithme en pseudo-code pour le jeu "Devine le nombre" :


ALGORITHME DevineLeNombre
VARIABLES
  nombre_secret: ENTIER
  essai: ENTIER
CONSTANTES
  MIN: 1
  MAX: 100

DEBUT
  nombre_secret <-- ALEATOIRE(MIN, MAX)
  REPETER
    LIRE essai
    SI essai > nombre_secret ALORS
      AFFICHER "Trop grand"
    SINON SI essai < nombre_secret ALORS
      AFFICHER "Trop petit"
    FIN SI
  JUSQU'A essai == nombre_secret
  AFFICHER "Gagné !"
FIN

Structures de Données Fondamentales

Les structures de données permettent d'organiser et de stocker les informations.

La File (FIFO - First In, First Out)

Dans une file, le premier élément ajouté est le premier à être retiré. C'est comme une file d'attente.


# En Python, on peut utiliser le module `collections.deque`
from collections import deque
file = deque(['a', 'b', 'c'])
file.append('d')  # Ajoute à la fin
print(file.popleft()) # Retire et renvoie 'a'

La Pile (LIFO - Last In, First Out)

Dans une pile, le dernier élément ajouté est le premier à être retiré. C'est comme une pile d'assiettes.


# Une liste Python peut être utilisée comme une pile
pile = ['a', 'b', 'c']
pile.append('d') # Ajoute à la fin
print(pile.pop()) # Retire et renvoie 'd'

Implémentation en Python

Traduire un algorithme en Python est une étape essentielle. Voici l'implémentation du jeu "Devine le nombre" :


import random

nombre_secret = random.randint(1, 100)
essai = 0

while essai != nombre_secret:
    try:
        essai = int(input("Devinez le nombre (entre 1 et 100) : "))
        if essai > nombre_secret:
            print("Trop grand !")
        elif essai < nombre_secret:
            print("Trop petit !")
    except ValueError:
        print("Veuillez entrer un nombre valide.")

print("Bravo, vous avez trouvé le nombre !")
