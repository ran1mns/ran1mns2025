Titre du Cours : Introduction aux Réseaux et Connexions

Introduction

Ce cours vous introduira aux concepts fondamentaux des réseaux et des connexions. Vous apprendrez les différentes technologies de connexion, les types de fibres optiques, l'évolution du Wi-Fi, le recyclage des équipements électroniques, et les modèles OSI et TCP/IP. Ces sujets sont essentiels pour comprendre comment les réseaux fonctionnent et comment les données sont transmises.

Concepts Clés

Types de Connexions Réseau

Les connexions réseau peuvent être établies via différentes technologies : Ethernet, Bluetooth, Wi-Fi, et fibre optique. Chacune a ses propres avantages et inconvénients.

Ethernet

Ethernet est une technologie de réseau filaire utilisée pour connecter des ordinateurs et d'autres dispositifs dans un réseau local (LAN). Elle est rapide et fiable, mais nécessite des câbles physiques.

Bluetooth

Bluetooth est une technologie sans fil utilisée pour connecter des dispositifs sur de courtes distances. Elle est couramment utilisée pour les périphériques comme les écouteurs, les claviers et les souris.

Wi-Fi

Le Wi-Fi permet de connecter des dispositifs à un réseau sans fil. Il est largement utilisé pour les réseaux domestiques et professionnels. Le Wi-Fi a évolué à travers plusieurs générations, chacune offrant des débits de données plus élevés et des plages de fréquences différentes.

Fibre Optique

La fibre optique utilise des fils de verre ou de plastique pour transmettre des données sous forme de lumière. Il existe deux types principaux de fibres optiques : la fibre monomode et la fibre multimode.

  • Fibre Monomode : Utilise une technologie laser pour transmettre des données sur de longues distances avec une faible atténuation. Le diamètre du cœur est d'environ 5 microns.
  • Fibre Multimode : Utilise une technologie LED pour transmettre des données sur des distances plus courtes. Le diamètre du cœur est d'environ 50 microns.

Évolution du Wi-Fi

Le Wi-Fi a évolué à travers plusieurs générations, chacune offrant des débits de données plus élevés et des plages de fréquences différentes.

Génération Débit Maximal Plage de Fréquences
Wi-Fi 4 (802.11n) 600 Mbps 2.4 GHz et 5 GHz
Wi-Fi 5 (802.11ac) 3.5 Gbps 5 GHz
Wi-Fi 6 (802.11ax) 9.6 Gbps 2.4 GHz et 5 GHz

Note : Plus la fréquence est élevée, plus on peut transmettre de données, mais moins les ondes traversent les obstacles comme les murs.

Recyclage des Déchets d'Équipements Électriques et Électroniques (DEEE)

Le recyclage des DEEE est obligatoire pour les producteurs, qui doivent organiser la filière de recyclage. Cependant, le consommateur paye une taxe pour financer ce recyclage.

Avertissement : Il est crucial de recycler les équipements électroniques pour réduire l'impact environnemental et récupérer les matériaux précieux.

Modèle OSI

Le modèle OSI (Open Systems Interconnection) est un cadre conceptuel utilisé pour comprendre et implémenter un réseau en couches. Il est composé de sept couches :

  1. Couche Physique : Transmission des bits sur un support physique comme un câble.
  2. Couche Liaison de Données : Transmission des trames entre deux nœuds adjacents.
  3. Couche Réseau : Routage des paquets de données entre les réseaux.
  4. Couche Transport : Transfert des données entre les applications.
  5. Couche Session : Gestion des sessions de communication.
  6. Couche Présentation : Formatage des données pour le stockage ou l'envoi.
  7. Couche Application : Interface avec les applications réseau comme les sites web.

Astuce : Les couches 1 à 4 sont généralement implémentées en hardware, tandis que les couches 5 à 7 sont implémentées en software.

Modèle TCP/IP

Le modèle TCP/IP est un autre cadre de référence pour les réseaux, composé de quatre ou cinq couches :

  1. Couche Accès Réseau : Correspond aux couches Physique et Liaison de Données du modèle OSI.
  2. Couche Internet : Correspond à la couche Réseau du modèle OSI.
  3. Couche Transport : Correspond à la couche Transport du modèle OSI.
  4. Couche Application : Correspond aux couches Session, Présentation et Application du modèle OSI.

Adresses et Identifiants

Les adresses et identifiants sont essentiels pour la communication réseau.

  • Adresse Physique (MAC) : Identifiant unique de 48 bits attribué à chaque interface réseau. Il est composé de 24 bits pour le numéro du constructeur et 24 bits pour le numéro de série.
  • Adresse Réseau (IP) : Adresse logique utilisée pour identifier les dispositifs sur un réseau. IPv4 utilise 32 bits, tandis qu'IPv6 utilise 128 bits.
  • Identifiant de la Couche Transport (Port) : Numéro de 16 bits utilisé pour identifier les applications. Il y a 65534 possibilités.

Appareils Réseau

Différents appareils sont utilisés pour gérer et interconnecter les réseaux :

  • Hub (Concentrateur) : Appareil de base pour interconnecter des dispositifs dans un réseau local. Il n'est plus couramment utilisé.
  • Switch (Commutateur) : Appareil plus avancé qui remplace le hub en utilisant la commutation pour diriger les données de manière plus efficace.
  • Routeur : Appareil utilisé pour interconnecter différents réseaux et diriger les paquets de données entre eux.
  • Firewall (Pare-feu) : Appareil ou logiciel utilisé pour protéger un réseau contre les accès non autorisés.
