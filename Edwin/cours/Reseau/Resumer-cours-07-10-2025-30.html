Titre du Cours : Adresse IP et DNS

Introduction

Les adresses IP et le DNS (Domain Name System) sont des éléments fondamentaux du fonctionnement d'Internet. Comprendre ces concepts vous permettra de mieux appréhender comment les réseaux informatiques fonctionnent et comment les sites web sont accessibles. Ce cours vous guidera à travers les bases des adresses IP, leur fonctionnement, et l'importance du DNS.

Concepts Clés

Qu'est-ce qu'une adresse IP ?

Une adresse IP (Internet Protocol) est une adresse unique attribuée à chaque appareil connecté à un réseau utilisant le protocole Internet. Elle permet d'identifier et de localiser ces appareils. Chaque équipement dans un réseau a une adresse IP unique, et deux équipements ne peuvent pas avoir la même adresse IP en même temps.

Astuce : Vous pouvez trouver l'adresse IP de votre appareil en utilisant la commande ipconfig dans l'invite de commande (CMD) sur Windows.

Types d'adresses IP

Il existe deux types principaux d'adresses IP : statiques et dynamiques.

  • Adresse IP statique : Une adresse IP fixe qui ne change pas. Elle est souvent utilisée pour les serveurs ou les équipements réseau qui doivent être accessibles en permanence.
  • Adresse IP dynamique : Une adresse IP qui peut changer à chaque connexion au réseau. Elle est attribuée automatiquement par un serveur DHCP (Dynamic Host Configuration Protocol).

Utilisation de la commande CMD

La commande ipconfig dans l'invite de commande (CMD) sur Windows permet de voir les informations réseau de votre appareil, y compris l'adresse IP.


ipconfig

L'adresse IPv4 est l'adresse IP attribuée par le réseau pour se connecter à Internet. La passerelle par défaut est l'adresse IP du routeur côté interne.

Fonctionnement du routeur

Le routeur utilise son adresse publique pour satisfaire les requêtes sur Internet. Il gère les flux de demandes et de réponses entre les appareils du réseau local et Internet.

Commande Ping

La commande ping est utilisée pour vérifier si une adresse IP ou un nom de domaine est accessible. C'est la plus petite demande possible pour voir si une adresse répond.


ping google.com

En pingant google.com, vous pouvez voir l'adresse IP du serveur où Google est hébergé.

Les plages d'adresses IP

Une adresse IPv4 est composée de 4 chiffres, chacun allant de 0 à 255, séparés par des points. Par exemple, 192.168.1.1. Chaque chiffre représente un octet, soit 8 bits en binaire.


00000000 = 0
11111111 = 255

Un octet (ou byte en anglais) correspond à 8 positions binaires, allant de 00000000 (0) à 11111111 (255).

Explications sur les disques durs et CD-ROM

Les disques durs et les CD-ROM sont des dispositifs de stockage de données. Les données sont écrites et stockées en octets (bytes). La lecture de ces données se fait en interprétant les bits stockés.

Astuce : Pour en savoir plus sur la représentation binaire des données, consultez la table ASCII qui associe chaque caractère à une valeur binaire.
