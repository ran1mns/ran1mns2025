Titre du Cours : Au Cœur de l'Ordinateur : Composants et Périphériques

Introduction

Ce cours vous plonge à l'intérieur d'un ordinateur pour en découvrir les composants essentiels. Nous verrons comment ils fonctionnent ensemble, du processeur au disque dur, et comment ils se connectent aux périphériques que nous utilisons tous les jours.

1. Le Cœur de l'Ordinateur

Ces trois composants forment le trio essentiel de tout ordinateur.

  • La Carte Mère (Motherboard) : C'est le circuit principal qui interconnecte tous les composants (CPU, RAM, stockage, etc.) via des "bus" de communication.
  • Le Processeur (CPU - Central Processing Unit) : C'est le "cerveau" de l'ordinateur. Il exécute les instructions et effectue les calculs. Les principaux fabricants sont Intel et AMD.
  • La Mémoire Vive (RAM - Random Access Memory) : C'est une mémoire ultra-rapide mais volatile (elle s'efface quand l'ordinateur est éteint). Elle stocke temporairement les données des programmes en cours d'exécution.

2. Le Stockage : Garder les Données

Contrairement à la RAM, le stockage est non-volatile : il conserve les données même hors tension.

TypeTechnologieAvantagesInconvénients
HDD (Hard Disk Drive) Mécanique (plateaux magnétiques qui tournent) Grande capacité, faible coût. Lent, fragile, bruyant.
SSD (Solid-State Drive) Électronique (puces de mémoire flash) Très rapide, silencieux, résistant. Plus cher, durée de vie limitée en écriture.

Technologie RAID

RAID (Redundant Array of Independent Disks) combine plusieurs disques pour améliorer soit la performance, soit la sécurité, soit les deux.

  • RAID 0 (Entrelacement) : Les données sont réparties sur plusieurs disques. Très rapide, mais si un disque tombe en panne, tout est perdu.
  • RAID 1 (Miroir) : Les données sont copiées à l'identique sur deux disques. Très sûr, mais on "perd" la moitié de l'espace de stockage total.
  • RAID 5 (Parité répartie) : Nécessite au moins 3 disques. Combine performance et sécurité. Peut survivre à la panne d'un disque.

3. La Connectique : Brancher les Périphériques

USB (Universal Serial Bus)

Le connecteur universel pour brancher claviers, souris, clés USB, etc. Il a beaucoup évolué :

VersionVitesse ThéoriqueUsage Courant
USB 2.0480 MbpsClaviers, souris.
USB 3.0 (ou 3.1 Gen 1)5 GbpsDisques durs externes.
USB 3.1 Gen 210 GbpsSSD externes rapides.
USB-CVariable (jusqu'à 40 Gbps)Connecteur réversible moderne pour tout type de périphérique.

Connecteurs Vidéo

  • HDMI : Le standard actuel pour la plupart des écrans et téléviseurs.
  • DisplayPort : Très populaire pour les moniteurs d'ordinateur, surtout pour les hautes fréquences de rafraîchissement.
  • VGA/DVI : Anciens connecteurs, de moins en moins utilisés.

4. Les Périphériques

Ce sont tous les appareils que l'on connecte à l'unité centrale.

  • Périphériques d'entrée : Clavier, souris, scanner, webcam, microphone.
  • Périphériques de sortie : Écran, imprimante, haut-parleurs, vidéoprojecteur.
  • Périphériques d'entrée/sortie : Écran tactile, disque dur externe, imprimante multifonction.

Exercices Pratiques

  1. Analysez votre ordinateur : Ouvrez les informations système de votre ordinateur (ou utilisez un logiciel comme Speccy) et identifiez : le modèle de votre CPU, la quantité de RAM, et le type de votre disque de stockage principal (HDD ou SSD).
  2. Construisez une configuration virtuelle : Sur un site de vente de composants (ex: LDLC, TopAchat), créez une configuration d'ordinateur complète pour un budget de 800€. Justifiez vos choix pour la carte mère, le CPU, la RAM et le stockage.
