Titre du Cours : L'ordinateur, ses composants et ses périphériques

Introduction

Ce cours vous introduira aux composants essentiels d'un ordinateur et à leurs périphériques. Vous apprendrez comment ces éléments interagissent pour faire fonctionner un ordinateur et comprendrez leur importance dans le monde de l'informatique.

Concepts Clés

La Carte Mère et ses Périphériques

La carte mère est le composant central de l'ordinateur. Elle est entourée de divers périphériques qui lui sont connectés. Voici un schéma simplifié des principaux composants :

Le Processeur (CPU)

Le CPU (Central Processing Unit) est le cerveau de l'ordinateur. Il exécute les instructions des programmes.

La Mémoire

Il existe deux types principaux de mémoire :

  • RAM (Random Access Memory) : Mémoire volatile qui perd ses données lorsqu'elle n'est plus alimentée.
  • ROM (Read Only Memory) : Mémoire non volatile qui conserve ses données même sans alimentation.

La RAM peut être dynamique (DRAM) ou statique (SRAM). La DRAM est plus courante et moins coûteuse, mais elle nécessite un rafraîchissement périodique.

Les Bus

Les bus sont des canaux de communication qui relient les différents composants de l'ordinateur. Il existe trois types de bus principaux :

  • Bus d'adresse : Transmet les adresses mémoire.
  • Bus de données : Transmet les données entre les composants.
  • Bus de commande : Transmet les signaux de commande.

Processeurs et Sockets

Les processeurs sont principalement fabriqués par Intel et AMD. Chaque série de processeurs utilise un socket spécifique :

  • PGA (Pin Grid Array) : Socket avec des broches.
  • LGA (Land Grid Array) : Socket avec des contacts plats.

Le refroidissement est crucial pour éviter la surchauffe. Les solutions courantes incluent le refroidissement par air et le watercooling.

La Carte Mère

La carte mère est composée de deux parties principales :

  • South Bridge : Gère les périphériques lents.
  • North Bridge : Gère les périphériques rapides et est proche du CPU.

Note : Les technologies MEMS (Micro Electro Mechanical Systems) et nanotechnologies sont des domaines de recherche avancés dans le développement des composants informatiques.

La Mémoire RAM

La mémoire DDR (Double Data Rate) a évolué au fil des ans :

  • DDR2, DDR3, DDR4, DDR5 : Chaque nouvelle série augmente la vitesse, optimise la consommation d'énergie, améliore la correction d'erreurs (ECC) et augmente la capacité par module.

Pour les ordinateurs portables, on utilise des modules SO-DIMM DDR de dimensions réduites.

Les Disques Durs

Il existe deux types principaux de disques durs :

  • HDD (Hard Disk Drive) : Disque dur mécanique avec des têtes de lecture et des disques rotatifs. Plus lent mais permet la récupération des données.
  • SSD (Solid State Disk) : Disque électronique rapide mais les données ne sont pas récupérables en cas de panne.

Connectique

Les interfaces de connexion ont évolué pour supporter des vitesses de transfert plus élevées :

  • ATA (Advanced Technology Attachment) : Interface parallèle.
  • SATA (Serial ATA) : Interface série avec des câbles différents et des vitesses de transfert plus élevées.

Les nouvelles connectiques incluent M.2 SATA et M.2 NVMe pour les SSD, avec différentes tailles (22110, 2280, etc.).

Technologie RAID

RAID (Redundant Array of Independent Disks) est une technologie de stockage qui combine plusieurs disques pour améliorer les performances et la fiabilité :

  • RAID 0 : Striping (améliore les performances).
  • RAID 1 : Mirroring (améliore la fiabilité).
  • RAID 5 : Striping avec parité (améliore les performances et la fiabilité).
  • RAID 6 : Striping avec double parité (améliore encore la fiabilité).
  • RAID 10 : Combinaison de RAID 1 et RAID 0 (mirroring et striping).

Conseil : Consultez des vidéos sur YouTube et des tableaux comparatifs pour mieux comprendre les différents niveaux de RAID.

Les Connecteurs

Les connecteurs à l'arrière de la carte mère permettent de connecter divers périphériques. Voici quelques exemples :

  • VGA (Video Graphics Array) : Connexion analogique.
  • DVI (Digital Visual Interface) : Connexion analogique et numérique.
  • DisplayPort : Connexion numérique, permet le branchement en série.
  • HDMI (High-Definition Multimedia Interface) : Connexion numérique, existe en versions mini et micro.

La Carte Graphique

La mémoire de la carte graphique (GDDR6) est plus performante que la mémoire DDR classique. Les connectiques ont évolué :

  • VGA : Analogique.
  • DVI : Analogique et numérique.
  • DisplayPort et HDMI : Numérique.

La résolution des écrans a également évolué, passant de 640x480 en VGA à 7680x4320 en Ultra HD 8K.

Les Cartes Son

Les cartes son gèrent les entrées et sorties audio. Les connectiques incluent :

  • RCA : Connexion audio classique.
  • Toslink : Connexion optique.
  • MIDI (Musical Instrument Digital Interface) : Connexion pour instruments de musique.

Le Port USB

Le port USB (Universal Serial Bus) est utilisé pour connecter une variété de périphériques. Les types de connecteurs incluent :

  • Type A : Connecteur standard.
  • Type B : Connecteur pour périphériques.
  • Mini A et Mini B : Connecteurs plus petits.
  • Type C : Connecteur réversible en vogue.

Périphériques de Stockage

Les périphériques de stockage incluent :

  • Clé USB : Stockage portable.
  • Cartes graphiques externes : Pour améliorer les performances graphiques.
  • Cartes réseaux : Connecteurs RJ45 pour le réseau.
  • Cartes modem : Pour la connexion internet.

Alimentation

L'alimentation fournit l'énergie nécessaire aux composants de l'ordinateur. Les lecteurs et enregistreurs externes incluent :

  • Disquettes : Ancien support de stockage.
  • Bandes magnétiques : Enregistrement séquentiel, plus sécurisé.
  • CD, DVD, Blu-Ray : Supports optiques.

Périphériques Divers

D'autres périphériques incluent :

  • Tablettes et tablettes graphiques : Pour le dessin et la navigation.
  • Imprimantes : Matricielles, jet d'encre, thermique, laser.
  • Scanners et traceurs : Pour la numérisation et l'impression de grande taille.
  • Imprimantes 3D : Pour la fabrication d'objets.
  • Caméras, vidéo, audio : Pour la capture et la diffusion.
  • Visioconférence : Pour les communications vidéo.
  • Onduleurs : Pour protéger contre les coupures de courant.
