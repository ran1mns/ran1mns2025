Titre du Cours : Commandes Linux de Base

Introduction

Les commandes Linux sont essentielles pour interagir avec le système d'exploitation Linux. Elles permettent de créer des répertoires, installer des programmes, naviguer dans le système de fichiers, et bien plus encore. Ce cours vous guidera à travers les commandes de base et leur utilisation pratique.

Concepts Clés

Syntaxe des Commandes

La syntaxe des commandes Linux peut varier, mais elle suit généralement l'une des formes suivantes :


nom-commande [-option]
nom_commande
commande [parametre]
commande [-option] [parametre]

Commandes de Base

Créer un Répertoire

La commande mkdir (make directory) permet de créer un nouveau répertoire.


mkdir nom_du_repertoire

Lister les Fichiers et Répertoires

La commande ls (list) permet de lister les fichiers et répertoires dans le répertoire courant.


ls

Pour un affichage détaillé, utilisez l'option -l :


ls -l

Afficher le Chemin Courant

La commande pwd (print working directory) affiche le chemin absolu du répertoire courant.


pwd

Naviguer dans le Système de Fichiers

La commande cd (change directory) permet de changer de répertoire. Utilisée seule, elle vous ramène à votre répertoire home.


cd nom_du_repertoire
cd ..  # Remonter d'un répertoire

Supprimer un Répertoire

La commande rmdir (remove directory) permet de supprimer un répertoire vide.


rmdir nom_du_repertoire

Combiner les Commandes

Vous pouvez combiner plusieurs commandes pour créer une arborescence de répertoires :


mkdir travail/compta travail/achat travail/achat/2024 travail/achat/2025

Installer et Gérer des Programmes

La commande apt (Advanced Packaging Tool) est utilisée pour installer et gérer des programmes sur les distributions basées sur Debian (comme Ubuntu).


apt update  # Met à jour la liste des applications disponibles

Déplacer et Renommer des Fichiers

La commande mv (move) permet de déplacer ou renommer des fichiers et répertoires.


mv ancien_nom nouveau_nom  # Renomme un fichier ou répertoire

Créer un Fichier Vide

La commande touch permet de créer un fichier vide.


touch nom_du_fichier

Supprimer des Fichiers et Répertoires

La commande rm (remove) permet de supprimer des fichiers.


rm nom_du_fichier

Pour supprimer un répertoire et tout son contenu, utilisez l'option -r :


rm -r nom_du_repertoire

Architecture 3 Tiers

L'architecture 3 tiers est une méthode de conception logicielle qui sépare les applications en trois couches :

  1. Présentation : Comment les choses sont présentées à l'utilisateur (navigateur web).
  2. Traitement : Couche métier, algorithmes, serveur web.
  3. Données : Bases de données.

Les requêtes entre les couches se font via HTTP et HTTPS sur les ports 80 et 443. Le serveur web cherche les fichiers et les renvoie au navigateur. Le langage PHP permet de générer le contenu de la page dynamiquement.

Le système LAMP (Linux, Apache, MySQL, PHP) est un exemple d'architecture 3 tiers.

Exercices Pratiques

  1. Créer une arborescence de répertoires pour un projet fictif avec les répertoires suivants : projet/docs, projet/src, projet/tests.
  2. Naviguer dans cette arborescence et lister les fichiers et répertoires à chaque niveau.
  3. Créer un fichier vide nommé README.md dans le répertoire projet/docs.
  4. Renommer le répertoire tests en unit_tests.
  5. Supprimer le répertoire unit_tests et tout son contenu.
Corrigé 

# Exercice 1
mkdir projet/docs projet/src projet/tests

# Exercice 2
cd projet
ls
cd docs
ls
cd ../src
ls
cd ../tests
ls

# Exercice 3
touch docs/README.md

# Exercice 4
mv tests unit_tests

# Exercice 5
rm -r unit_tests
