Titre du Cours : Introduction à l'Hébergement Web et aux Serveurs

Introduction

L'hébergement web est une étape cruciale dans le développement et la mise en ligne de sites internet. Il consiste à louer un espace sur un serveur pour y stocker les fichiers de votre site web. Ce cours vous guidera à travers les différentes options d'hébergement, les configurations de serveurs, et les bases de la gestion de serveurs sous Linux.

Concepts Clés

Hébergement Web

L'hébergement web est un service qui permet de stocker les fichiers de votre site web sur un serveur accessible via Internet. Il existe plusieurs types d'hébergement, chacun avec ses avantages et inconvénients.

Hébergement Mutualisé

L'hébergement mutualisé est une place préconfigurée sur un serveur partagé avec d'autres sites. C'est une option peu coûteuse où vous êtes responsable uniquement de votre site, tandis que tous les autres aspects sont gérés par l'hébergeur. Cependant, il est limité en termes de langages de programmation et de personnalisation.

VPS (Virtual Private Server)

Un VPS est une machine virtuelle dédiée où vous avez plus de contrôle sur la configuration du serveur. Vous pouvez choisir les caractéristiques techniques (processeur, RAM, disque dur) en fonction de vos besoins. C'est une option intermédiaire entre l'hébergement mutualisé et le serveur dédié.



# Configuration de base d'un VPS
hostname myvps
memory 2048
cpus 2

Serveur Dédié

Un serveur dédié est une machine physique entièrement à votre disposition. C'est l'option la plus personnalisable mais aussi la plus coûteuse. Vous avez un contrôle total sur les configurations matérielles et logicielles.

Hyperviseurs

Les hyperviseurs sont des logiciels qui permettent de créer et de gérer des machines virtuelles. Ils existent en deux types principaux :

Hyperviseur de Type 2

Les hyperviseurs de type 2, comme VirtualBox, sont utilisés principalement pour des tests et des environnements de développement. Ils s'exécutent sur un système d'exploitation hôte.

Hyperviseur de Type 1

Les hyperviseurs de type 1, comme VMware ESXi, sont des systèmes d'exploitation à part entière dédiés à la gestion de machines virtuelles. Ils sont plus performants et utilisés dans des environnements professionnels.

Réseaux et Ports

La configuration réseau est essentielle pour permettre l'accès à votre serveur. Voici quelques ports couramment utilisés :

  • HTTP : Port 80
  • HTTPS : Port 443
  • FTP : Port 22

Vous pouvez configurer votre routeur pour la redirection de ports afin de contrôler quelles informations passent selon les protocoles.



# Redirection de port dans un fichier de configuration de routeur
port_forwarding {
  from_port = 80
  to_port = 8080
  protocol = tcp
}

Gestion de Serveurs sous Linux

Sous Linux, la gestion des fichiers et des répertoires est différente de celle sous Windows. Linux ne gère pas les extensions de fichiers de la même manière.

Types de Fichiers

Linux identifie les fichiers en tant que "répertoire", "périphérique", ou "normal" (texte, vidéo, etc.). Nous nous intéresserons principalement aux fichiers répertoire et normal.

Commandes de Base

Voici quelques commandes de base pour gérer votre serveur sous Linux :

  • clear : Nettoie l'écran de la console.
  • apt update : Met à jour la liste des programmes disponibles sur les serveurs Debian.


# Nettoyer l'écran
clear

# Mettre à jour la liste des programmes
apt update

Sécurité des Mots de Passe

La robustesse des mots de passe est cruciale pour la sécurité de votre serveur. Utilisez des mots de passe complexes et suivez les recommandations de l'ANSSI (KeePass).

Astuce : Technique du Texte de Base

Une méthode pour créer des mots de passe robustes est d'utiliser un texte de base, un numéro de ligne, et une méthode pour créer le mot de passe. Par exemple, utilisez une fable de La Fontaine.

Connexion SSH

SSH (Secure Shell) est un protocole sécurisé pour se connecter à distance à votre serveur. Il utilise un échange de clés privée et publique pour authentifier la connexion.



# Connexion SSH
ssh user@hostname
