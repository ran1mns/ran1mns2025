Organisation générale

Durée : 5 séances (demi-journées)

Objectifs :

  • Être attentif à la prise de notes pour mieux retenir l’information
  • Trouver sa propre méthode de travail et d’organisation (logiciels recommandés : Obsidian, Notion…)

Présentation du formateur :

Démonstration d’Obsidian, logiciel de prise de notes basé sur le concept de "graphes de connaissances", permettant de lier les idées et de structurer l’information.

Focus : Obsidian

Obsidian est un outil de prise de notes qui crée un graphe de connaissances. Chaque note peut être liée à d’autres, permettant de visualiser les relations entre idées et faciliter la révision.

1. Bases de la culture informatique

1.1 Matériel et logiciel

  • Matériel (hardware) : composants physiques de l’ordinateur
  • Logiciel (software) : programmes permettant de faire fonctionner le matériel
  • Systèmes d’exploitation : gestion du matériel et des logiciels (ex. Windows, Linux, macOS)
  • Logiciels applicatifs : programmes pour l’utilisateur (ex. Word, Excel, Photoshop)
Utilisateur → Application → Système d’exploitation → Matériel

Focus : Système d’exploitation

Le système d’exploitation est le logiciel central qui permet aux programmes de communiquer avec le matériel. Il gère la mémoire, les périphériques et fournit une interface pour l’utilisateur.

2. L’algorithme

Définition : une série d’instructions pour résoudre un problème ou accomplir une tâche.

Historique :

  • ~3000 av. J.-C. : premières procédures et recettes sur tablettes
  • 9ᵉ siècle : Al-Khwarizmi, père de l’algorithme et de l’algèbre
  • 19ᵉ siècle : Ada Lovelace écrit des algorithmes pour la machine analytique de Babbage
  • 20ᵉ siècle : Alan Turing formalise la notion d’algorithme

Focus : Algorithme

Un algorithme transforme des données d’entrée en résultats. Il peut être représenté en pseudo-code ou diagramme et sert de base à tous les programmes informatiques.

3. Histoire des calculateurs

3.1 Calculateurs mécaniques

  • Boulier chinois (~2000 av. J.-C.)
  • Machine d’Anticythère (~200 av. J.-C.)
  • Schickard : horloge calculante (1592-1635)
  • Pascal : Pascaline (1623-1662)
  • Babbage : machine analytique (1791-1871)
  • Ada Lovelace : première programmation (1815-1852)

Focus : Machine analytique

Première conception d’ordinateur programmable. Contient unité de calcul, mémoire et cartes perforées pour instructions.

3.2 Calculateurs électromécaniques

  • Relais électromécaniques (0/1) (début 20ᵉ siècle)
  • Konrad Zuse : Zuse 3 (1941)
  • Enigma (Scherbius, 1920s-1930s)
  • Alan Turing : déchiffrement d’Enigma (1940-1945)

Focus : Enigma

Machine de chiffrement allemande, déchiffrée par Turing, influence majeure sur l’informatique et la cryptographie moderne.

3.3 Calculateurs électroniques

  • Tubes à vide (1945-1955)
  • ENIAC (1945) : premier ordinateur électronique à usage général
  • UNIVAC I (1951)
  • FERRANTI MARK I (1951)
  • BINAC (1949)

Focus : ENIAC

Premier ordinateur électronique à usage général, très rapide comparé aux calculateurs précédents. Utilisé pour calculs scientifiques et militaires.

3.4 Transistors et microprocesseurs

  • Transistors (1955-1970) : miniaturisation, moins de chaleur, plus rapide que les tubes à vide
  • MICRAL (1972) : processeur Intel 8008
  • Intel 4004 (1971) : premier microprocesseur
  • Altair 8800 (1975) : micro-ordinateur populaire
  • Apple I (1976) : ordinateur personnel

Focus : Microprocesseur

Unité centrale miniaturisée qui exécute les instructions d’un ordinateur. Base des ordinateurs personnels modernes.

4. Les logiciels

  • Traitement de texte : Word (1983), LibreOffice Writer (2002)
  • Tableurs : Excel (1985), Google Sheets (2006)
  • Bases de données : Access (1992), MySQL (1995), Oracle (1979), PostgreSQL (1986)
  • Logiciels graphiques : Photoshop (1990), GIMP (1996)
  • Navigateurs web : Netscape (1994), Chrome (2008), Firefox (2002)

5. Historique des machines

  • 1940-1950 : Ordinateurs de première génération (tubes à vide)
  • 1955-1965 : Deuxième génération (transistors)
  • 1965-1975 : Troisième génération (circuits intégrés)
  • 1975-1985 : Quatrième génération (microprocesseurs)
  • 1985 à aujourd'hui : Cinquième génération (ordinateurs personnels et mobiles, IA)

Focus : Évolution

Chaque génération a apporté une miniaturisation, une augmentation de vitesse et une baisse des coûts, rendant les ordinateurs accessibles au grand public.

6. Téléphonie

  • Téléphone fixe : invention d’Alexander Graham Bell (1876)
  • Téléphone mobile : apparition des premiers téléphones portables (1973)
  • Téléphonie IP : communication via Internet (VoIP)
  • Smartphones : combinaison de téléphone, ordinateur et capteurs divers

Focus : VoIP

La VoIP permet de passer des appels via Internet en utilisant des protocoles comme SIP ou WebRTC, réduisant les coûts et permettant la mobilité.

7. Systèmes d’exploitation

  • Windows : interface graphique, utilisé sur la plupart des PC
  • Linux : open-source, multi-usage (serveurs, PC, embarqué)
  • macOS : basé sur Unix, interface graphique et forte intégration matérielle
  • Android : système mobile basé sur Linux
  • iOS : système mobile Apple, optimisé pour iPhone et iPad

Focus : Open Source vs Propriétaire

Linux est open-source et modifiable, tandis que Windows et macOS sont propriétaires et contrôlés par leurs éditeurs respectifs.

8. Internet & Sécurité

  • Internet : réseau mondial de communication, basé sur TCP/IP
  • HTTP/HTTPS : protocoles de communication pour le web
  • DNS : conversion des noms de domaine en adresses IP
  • Cyber-sécurité : protection des systèmes contre les intrusions, virus et attaques
  • Mots de passe : robustes, complexes et uniques pour chaque compte
  • VPN : réseau privé virtuel pour sécuriser la connexion et les données

Focus : Bonnes pratiques

Utiliser des mots de passe forts, mettre à jour logiciels et systèmes, sauvegarder les données, et privilégier les connexions sécurisées (HTTPS, VPN).
