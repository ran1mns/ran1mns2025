Ordinateur & périphériques

Carte mère entourée par les périphériques.

CPU / RAM / ROM / PIO → relié par bus d'adresse, bus de données et bus de commande
  • CPU : Central Processing Unit (unité centrale de traitement)
  • RAM : Random Access Memory, mémoire volatile (perdue sans énergie)
  • ROM : Read Only Memory, mémoire non volatile
  • PIO : Programmable Input/Output, interface entrée/sortie programmable

Focus : RAM

La RAM peut être dynamique (DRAM) ou statique (SRAM). La DRAM nécessite un rafraîchissement constant, la SRAM est plus rapide mais plus coûteuse. Elle est essentielle pour le fonctionnement des programmes en cours.

Processeurs & sockets

  • Intel et AMD : ~99% du marché
  • Sockets différents selon série :
    • PGA : Pin Grid Array
    • LGA : Land Grid Array
  • Chaleur : nécessité de refroidissement (air ou watercooling)

Focus : Socket

Un socket est l’interface physique entre le processeur et la carte mère. Chaque type de processeur nécessite un socket compatible.

La mémoire

  • DDR : Double Data Rate (transfert de données deux fois par cycle d'horloge)
  • Séries DDR2, DDR3, DDR4, DDR5 : vitesse et optimisation énergétique
  • ECC : Error-Correcting Code, mémoire capable de détecter et corriger certaines erreurs
  • Formats réduits pour laptop : SO-DIMM (Small Outline DIMM)

Focus : DDR5

DDR5 offre des vitesses supérieures, meilleure efficacité énergétique et supporte plus de mémoire par module. Essentielle pour les systèmes récents et applications lourdes.

Disques durs & SSD

  • HDD mécanique : plus lent, données récupérables (Hard Disk Drive)
  • SSD électronique : rapide, récupération limitée (Solid State Drive)
  • Connectiques : ATA (parallèle), SATA (série), M.2 SATA, M.2 NVMe
  • Technologie RAID : 0,1,5,6,10
  • Tailles M.2 : numéro = largeur × longueur en mm. Exemple : 2280 = 22 mm × 80 mm, 22110 = 22 mm × 110 mm

Focus : SSD NVMe

Disque SSD connecté via M.2 NVMe utilise le bus PCIe pour des vitesses très élevées, réduit les temps de lecture/écriture comparé au SATA.

Connectique

  • Diamant comme isolant et conducteur thermique performant
  • Connecteurs arrière carte mère : RJ45, audio, HDMI, DisplayPort…

Focus : M.2

Format compact pour SSD, supporte SATA et NVMe. Différentes longueurs adaptées aux cartes mères et laptops.

Carte graphique

  • RAM plus performante que la RAM classique (ex : GDDR6)
  • Connectiques : VGA, DVI, HDMI, DisplayPort
  • HDMI mini/micro, DisplayPort série possible
  • Résolutions : VGA 640x480 → Ultra HD 8K 7680x4320

Focus : GDDR6

RAM dédiée aux cartes graphiques, très rapide, optimise le rendu et les performances 3D par rapport à la RAM classique.

Cartes son

  • Entrées/sorties audio codées par couleurs
  • Connectiques : RCA, Toslink (fibre optique), MIDI DIN 5 broches

Focus : Toslink

Connectique audio numérique via fibre optique, haute fidélité et immunité aux interférences électriques.

Ports USB

  • Types : A, B, MiniA, MiniB, Type-C
  • Tableau date de sortie et vitesses : USB 1.0 (1996), 2.0 (2000), 3.0 (2008), 3.1/3.2, 4.0

Focus : USB Type-C

Connecteur réversible, supporte transfert rapide et alimentation, devient standard moderne pour périphériques et laptops.

Périphériques & multimédia

  • Clés USB, cartes graphiques externes, cartes réseau RJ45, cartes modem
  • Alimentations, lecteurs/enregistreurs externes (disquettes, bandes, CD/DVD/BluRay)
  • Tablettes, tablettes graphiques, imprimantes (matricielles, jet d’encre, laser, 3D)
  • Scanner, traceurs, caméras vidéo/audio, visioconférence, onduleurs

Focus : Imprimante 3D

Dispositif qui crée des objets en déposant couche par couche des matériaux. Utilisée en prototypage, industrie et éducation.
